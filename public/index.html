Todo API Documentation

Welcome to the Todo API! This API provides endpoints for user authentication and todo management.

View Swagger Documentation

Available Endpoints

POST /auth/register - Register a new user
POST /auth/login - Login user
GET /todo - Get all todos (requires authentication)
POST /todo - Create a new todo (requires authentication)

Quick Start

  1. Register a new user using /auth/register
  2. Login to get a JWT token using /auth/login
  3. Use the token in the Authorization header: Bearer YOUR_TOKEN
  4. Access protected endpoints like /todo
